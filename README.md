# Portfólio - Gabriel Henrique 🚀

Bem-vindo ao meu portfólio profissional! Este projeto foi desenvolvido para demonstrar minhas habilidades e competências como **Desenvolvedor Full Stack**.

## 🎯 Sobre o Projeto

O portfólio consiste em uma página web (Single Page Application estática) moderna, responsiva e interativa que apresenta meu perfil, formação, habilidades técnicas e os projetos práticos que desenvolvi.

### Tecnologias Utilizadas na Interface:
- **HTML5** com semântica moderna.
- **CSS3** (Variáveis, Flexbox, Grid, Glassmorphism, e animações nativas).
- **JavaScript (Vanilla)** (Efeitos de partículas no canvas, digitação dinâmica "typing", validações automáticas, Observer API para animações ao rolar a página).
- Ícones via **FontAwesome** e fontes otimizadas pelo Google Fonts (`Inter` e `JetBrains Mono`).

## 📁 Estrutura do Repositório

O repositório abriga tanto a interface do portfólio (raiz) quanto os códigos-fonte dos projetos demonstrados:

- `/index.html`, `/style.css`, `/script.js`: Arquivos principais do portfólio visual.
- `/gestao`: Sistema de Gestão completo desenvolvido com **Laravel (PHP)** e **SQLite**. Demonstra habilidades sólidas em back-end, arquitetura MVC, banco de dados e APIs. (Veja o README específico dentro dessa pasta para saber como rodar a aplicação Laravel).

## 🛠️ Minhas Competências Destacadas

Neste ecossistema demonstro domínio prático nas seguintes áreas:
- **Desenvolvimento Web:** React, Node.js, Laravel (PHP).
- **Lógica e Sistemas:** Criação de fluxos complexos no backend Laravel, validação de regras de negócio.
- **Integração:** Consumo e estruturação de APIs REST e testes funcionais automatizados (Jest/Pest/PHPUnit).
- **Banco de Dados:** Consultas SQL, manipulação via ORM (Eloquent) e migrations.
- **Inteligência Artificial:** Uso prático de IAs (Gemini, Claude, Antigravity) como aliadas para qualidade de código e produtividade.
